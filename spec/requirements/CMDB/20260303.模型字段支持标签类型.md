# 模型字段支持标签类型

日期：2026-03-03

## 背景

当前实例描述能力主要依赖固定字段，用户侧难以对数据进行低成本、灵活的语义标注（如环境、地域、业务属性）。

为下放数据描述权并提升实例检索与治理效率，CMDB 模型字段需支持内置“标签类型”字段，打通模型配置、实例录入/修改、列表展示、搜索、导入导出与批量更新链路。

## 需求描述

CMDB 模型字段新增内置标签能力（单模型单标签字段），标签以 `key:value` 键值对表达。

### 1）模型字段定义与启停规则

- 每个模型最多存在 1 个内置标签字段。
- 标签字段 ID 固定为 `__tag__`。
- 标签字段以开关形式启用/关闭：
  - 启用：标签字段出现在模型字段配置中，可进入编辑。
  - 关闭：后端物理删除该模型标签字段与关联标签数据；实例信息、实例列表、搜索、导入导出中不再出现标签字段。
- 模型字段列表中的标签字段操作（编辑/删除）需持续可见，不依赖 hover 才显示。
- 删除标签字段需二次确认。

### 2）标签字段编辑规则

- 字段 ID 固定为 `__tag__`，且不允许修改。
- 字段名称、属性分组允许修改。
- 内置约束固定且不允许修改：
  - 可编辑：是
  - 唯一：否
  - 必填：否
- 创建规则支持两种模式：
  - 自由模式：实例新增/修改时，允许用户输入并新增模型中尚未定义的标签项。
  - 严谨模式：实例新增/修改时，用户只能从模型已定义标签项中选择，不允许自定义输入。
- 标签项由 `key:value` 组成，支持同 key 多 value 并存（例如 `env:test`、`env:prod`、`env:uat`）。
- 标签 key 不允许为空。
- 标签 value 不允许包含空格、冒号、换行符。
- 单个实例最多设置 20 个标签对。
- 支持批量设置标签项（多行文本输入解析，每行一个键值对）。

### 3）实例新增/修改交互规则

- 标签输入控件需支持“可输入 + 可搜索选择”能力（Combo 输入）。
- 严谨模式下禁用自定义输入，仅允许从已定义候选项中选择（Select 行为）。
- 录入/修改阶段实时执行标签合法性校验：
  - 格式校验：必须为 `key:value`
  - key 校验：不可为空
  - value 字符校验：不可包含空格、冒号、换行符
  - 数量校验：最多 20 个标签对

### 4）实例列表展示与批量操作规则

- 实例列表支持展示标签列，展示格式为逗号分隔的 `key:value` 序列。
- 标签内容超长时自动截断，鼠标移入显示完整 tooltip。
- 列属性默认展示规则：第一列固定 `Name`，其后保留模型默认前 4 个字段；用户可调整并保留个人记忆。
- 选择实例后，`导出`、`批量更新`、`删除`按钮激活；未选择实例时保持禁用。

### 5）实例搜索规则（标签专用）

- 当搜索字段选择“标签”时，展示标签专用下拉筛选组件。
- 组件形态为二层树选择：
  - 第一层：标签 key 列表
  - 第二层：对应 key 的 value 列表
- 支持多选，已选内容在输入区域回显。
- 同一查询会话内支持跨页多选；切换筛选条件后清空已选集合。
- 标签搜索仅支持精确匹配。
- 标签搜索时 `Accurate` 固定为开启状态，不允许切换为模糊匹配。

### 6）导入/导出与批量更新规则

- Excel 中标签字段格式为 `key:value`。
- 多个标签对分隔符兼容：英文逗号（`,`）、中文逗号（`，`）、换行。
- 导入、导出、批量更新均复用同一标签校验规则：
  - key 不允许为空
  - value 不允许空格、冒号、换行符
  - 单实例最多 20 个标签对
  - 非法数据需返回可定位的错误信息（至少包含行号与字段名）

## 适用范围

- 模型字段定义与编辑
- 实例新增/修改
- 实例列表展示
- 实例搜索
- 实例导入/导出
- 实例批量更新

## 非目标（本期不包含）

- 标签层级扩展为三级及以上树结构
- 标签 value 的模糊检索能力
- 标签自动推荐、标签治理统计等运营能力
- 非标签类型字段的同步改造

## 验收标准

- 单模型可启用且仅可保留 1 个标签字段，且字段 ID 固定为 `__tag__`；关闭后标签字段与关联标签数据被物理删除，且实例相关页面与能力链路中不再出现标签信息。
- 标签字段编辑时，字段 ID 不可变更；名称与分组可变更；可编辑/唯一/必填内置约束不可改。
- 标签 key 为空时保存失败并提示。
- 自由模式下可新增自定义标签项；严谨模式下仅可从模型已定义标签项中选择。
- 同 key 多 value 可保存与回显；value 包含空格、冒号、换行符时保存失败并提示。
- 实例新增/修改、批量更新、导入均执行“最多 20 个标签对”与格式合法性校验。
- 标签列超长展示截断且可 tooltip 查看完整内容。
- 标签搜索展示专用二层筛选组件，支持多选回显与同查询会话跨页保留。
- 标签搜索仅精确匹配，`Accurate` 固定开启。
- Excel 导入导出支持 `key:value`，并兼容英文逗号、中文逗号、换行分隔。
